# PERGAMINO
Yet another lightweight node-based JSON editor used for writing dialogue and events in games.

![Pergamino preview image](assets/readme_preview.png)

Fully implemented in **Rust** using **egui** and **egui-snarl** by Adrián Alves (**alvesito**), as a tool used in the development of **RÓDRICA**.

Feel free to use it for your own purposes.
